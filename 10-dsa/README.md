## How to read these
Files 01-02 (Foundations) underpin everything after them. Files 03-10 (Data Structures) build on each other loosely but can mostly be read in order. Files 11-17 (Algorithms) lean on the data structures folder constantly. Files 18-19 tie everything together into actual interview strategy. Every file ends with a "Try this" exercise — DSA especially rewards actually solving problems, not just reading theory.
