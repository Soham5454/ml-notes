## How to read these
Files 01-04 (Linear Algebra), 05-08 (Calculus), 09-12 (Probability), and 13-16 (Statistics) each build directly on each other within their section. Every file links back to specific formulas/claims used loosely in the scikit-learn, XGBoost, and PyTorch folders — this folder is what makes those claims provable instead of just trusted. File 17 ties everything together with a full cross-reference map back to those folders.
