## How to read these
Files 01-02 cover the core request/response cycle. Files 03-05 build a real frontend (templates, forms, styling). File 06 connects a real trained model — the genuine milestone of this folder. Files 07-10 round out a production-grade app (REST APIs, structure, database, auth). File 11 covers deployment, and file 12 closes the loop into production ML serving tools (FastAPI, TorchServe) ahead of the MLOps folder.
