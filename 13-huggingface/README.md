# Hugging Face — A to Z

Notes on the Hugging Face ecosystem — the direct continuation of PyTorch (files reference back to that folder constantly) and the successor to the Classical NLP folder's pre-transformer techniques. Covers tokenizers, pipelines, datasets, fine-tuning across major NLP tasks, generation, evaluation, the Model Hub, and efficient fine-tuning with PEFT/LoRA.

## How to read these
Files 01-04 build the foundation (ecosystem, tokenization, inference, data loading). Files 05-07 apply the Trainer-based fine-tuning pattern to classification, token classification, and QA. Files 08-09 cover generation and Seq2Seq tasks. Files 10-12 round out evaluation, sharing, and efficient fine-tuning. File 13 closes the loop with a full concept map into the Frontier GenAI phase.
