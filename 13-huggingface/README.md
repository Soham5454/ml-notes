## How to read these
Files 01-04 build the foundation (ecosystem, tokenization, inference, data loading). Files 05-07 apply the Trainer-based fine-tuning pattern to three major task types. Every file links back to specific PyTorch and Classical NLP concepts throughout — this folder assumes both are already read.
