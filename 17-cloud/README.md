## How to read these
File 01 covers universal cloud concepts (IaaS/PaaS/SaaS, regions, elasticity) that apply across every provider. Files 02-06 go deep on AWS specifically — IAM, compute, storage, databases, and SageMaker — each explicitly cross-referencing the MLOps, PyTorch, Flask, and SQL folders' equivalent self-managed concepts. Files 07+ cover GCP, networking, cost, and infrastructure as code.
