Files 01-03 cover the LLM landscape and prompting skills needed before touching any code. File 04 covers the technical API foundation. Files 09-10 cover agents and multi-agent frameworks. Files 11-12 cover evaluation and safety. File 13 closes the loop into ML System Design.
