## How to read these
Files 01-06 build the complete design framework (requirements, data, features, modeling, serving) that every case study applies. Files 07-11 are fully-worked case studies across distinct problem types — recommendation, search, fraud, feed ranking, and RAG/GenAI. File 12 covers cross-cutting scale/reliability concerns and tradeoff communication, and file 13 bridges into final interview prep. Every file cross-references the entire roadmap — this folder is about communication, not new technical content.
