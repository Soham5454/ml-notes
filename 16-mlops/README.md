## How to read these
File 01 lays out the full ML lifecycle this folder fills in piece by piece. Files 02-03 cover containerization infrastructure. Files 04-06 cover tracking and versioning — reproducibility for experiments, data, and models. Files 07+ build on this foundation with automation, monitoring, and production deployment strategies.
