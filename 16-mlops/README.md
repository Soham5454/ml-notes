# MLOps — A to Z

Notes on MLOps — the discipline that turns everything built across the PyTorch, Hugging Face, and Flask folders into systems that run reliably over time, not just once in a demo. Covers containerization, experiment tracking, data/model versioning, CI/CD, monitoring, deployment strategies, and orchestration.

## How to read these
File 01 lays out the full ML lifecycle this folder fills in piece by piece. Files 02-03 cover containerization infrastructure. Files 04-06 cover tracking and versioning — reproducibility for experiments, data, and models. Files 07-08 automate testing and ML-specific quality gates. Files 09-11 cover what happens after deployment: monitoring, safe rollout strategies, and orchestration at scale. File 12 closes the loop into managed cloud equivalents.
